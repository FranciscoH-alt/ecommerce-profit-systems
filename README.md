# Ecommerce Profit Systems Portfolio

This repository contains four structured ecommerce analytics systems focused on profit clarity, contribution margin diagnostics, retention modeling, campaign performance analysis, and executive-level reporting.

Each project is self-contained and can be run independently.

## Repository Structure
ecommerce-profit-systems/
│
├── 01_project1_revenue_audit_engine/
├── 02_project2_ltv_cohort_analysis/
├── 03_project3_campaign_profit_diagnostics/
└── 04_project4_revenue_warehouse_dashboard/

## Projects Overview

### 1. Revenue Audit Engine
Order-level profitability diagnostics.
Focus: revenue integrity, margin leakage, contribution logic.

### 2. LTV Cohort Analysis
Customer retention and lifetime value modeling.
Focus: repeat purchase behavior, cohort performance.

### 3. Campaign Profit Diagnostics
Marketing spend efficiency analysis.
Focus: CAC, ROAS, break-even thresholds.

### 4. Revenue Warehouse with Executive Dashboard
Structured data warehouse and executive reporting layer.
Focus: KPI modeling, revenue visibility, executive decision support.

## How to Run Any Project

Navigate into the project folder:

    cd 01_project1_revenue_audit_engine

Create virtual environment:

    python3 -m venv .venv
    source .venv/bin/activate

Install dependencies:

    pip install -r requirements.txt

Run the main script:

    python <main_script>.py

(See individual project README for exact file names.)
